# Car Price Prediction using Linear Regression

This project focuses on predicting used car prices using the Linear Regression algorithm based on several vehicle features such as mileage, production year, engine volume, and brand. The project was developed using Python in Jupyter Notebook as part of a machine learning practice project.

## Project Objectives
- Analyze factors that influence used car prices
- Perform data preprocessing before modeling
- Build a Linear Regression model for price prediction
- Evaluate model performance using regression metrics

## Dataset
The dataset contains information about used cars with several features, including:
- Price
- Mileage
- Year
- EngineV
- Brand
- Body Type
- Engine Type

Target variable:
- Price

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
### 1. Data Preprocessing
- Handling missing values
- Removing outliers
- Log transformation
- Encoding categorical features
- Feature scaling

### 2. Exploratory Data Analysis (EDA)
Performed data visualization and analysis to understand:
- Relationship between year and price
- Relationship between mileage and price
- Relationship between engine volume and price
- Data distribution patterns

### 3. Model Building
- Train-test split
- Linear Regression modeling
- Price prediction

### 4. Model Evaluation
Model performance was evaluated using:
- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)

The evaluation results show that the model is able to predict used car prices reasonably well based on the available vehicle features.

## Business Insight
- Newer vehicles tend to have higher selling prices
- Higher mileage generally lowers vehicle prices
- Engine volume and brand influence car prices significantly
